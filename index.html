About me

Learn more about me

Hello, my name is Antonijo.

I am an energetic and imaginative web developer who creates beautiful, usable, professional websites and web applications with skills in HTML(5), CSS(3), Sass, Javascript (jQuery, JSON, React.js and Node.js), SQL, MongoDB and git(GitHub).

I have graduated with First Class Honours Degree in Higher Diploma in Science in Web Technologies at National College of Ireland.

Looking for an entry/junior position. I would love to combine my passion for learning with my web development skills to continue building an awesome websites and web applications in order to help people around us.

The Book Boutique

Description: Final project for Higher Diploma in Science in Web Technologies at National College of Ireland.

An online bookstore system is the main function of the web application where the user can check the book details, can register/login to buy the book, can add a book to wishlist or can add a book on the reading list.

Built with: Node.js, Javascript, Express, MongoDB, HTML, Handlebars, CSS(Sass), Bootstrap, Stripe
